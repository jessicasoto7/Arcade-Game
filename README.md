## Instructions

Get to the otherside before being attacked by the ladybugs.

You will start with 3 lives.

1 Ladybug attack = 2 lives left.

2 Ladybug attack = 1 life left.

3 Ladybug attack = 0 life left.

4 Ladybug attack = Game Over!
